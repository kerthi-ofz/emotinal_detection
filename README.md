# emotinal_detection
## Overview

Emotion Analysis System is a Deep Learning application that detects human facial emotions from images and real-time webcam video.

The system uses a CNN trained on the FER2013 dataset and OpenCV for face detection.

---

## Features


✔ Real-Time Webcam Detection

✔ Multiple Face Detection

✔ Confidence Score

✔ Emotion Detection

✔ Emotion Trend Graph

---

## Technologies Used

- Python
- TensorFlow/Keras
- OpenCV
- Streamlit
- NumPy
- Matplotlib

---

## Dataset

FER2013

Classes:

- Angry
- Happy
- Sad
- Neutral
- Surprise
- Fear
- Disgust
